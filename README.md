📧 Smart Irrigation System using Soil Moisture Monitoring and Automated Water Control

📖 Project Overview

This project is a smart farming system that automatically monitors soil moisture levels and controls water irrigation. It helps farmers reduce water wastage by turning the water pump ON/OFF based on real-time soil conditions using sensor data and automation logic.

🎯 Objectives

- Automate irrigation based on soil moisture level
- Reduce water wastage in agriculture
- Monitor real-time soil conditions
- Improve crop productivity using smart technology
- Build an efficient automated farming system

✨ Features

- Real-time soil moisture monitoring
- Automatic water pump ON/OFF control
- Live system dashboard
- Alert notification for dry soil
- Secure login system for users
- Data visualization and reports
- Fully automated irrigation process

🧩 System Modules

- Authentication Module – User login and access control
- Sensor Data Module – Collects soil moisture data
- Moisture Monitoring Module – Displays soil condition status
- Irrigation Control Module – Controls water pump automatically
- Dashboard Module – Shows system status and updates
- Notification Module – Sends alerts when soil is dry

🛠️ Technologies Used

- HTML, CSS, JavaScript
- Node.js / Python / PHP (Backend)
- MySQL / Firebase (Database)
- Soil Moisture Sensor
- Arduino / ESP32 Microcontroller
- Water Pump System

📊 System Workflow

Soil sensor collects moisture data → Data sent to system → System analyzes moisture level → Water pump turns ON/OFF automatically → Data displayed in dashboard and stored for tracking
